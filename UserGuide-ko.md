# 초급 가이드

_(시작에 필요한 모든 것)_

<a id="what-this-app-is-about"></a>

## 이 앱이 하는 일

Meditation & Breathwork는 명상과 호흡을 하는 동안 몸이 어떻게 반응하는지 부드럽게 관찰하면서 현재에 머물 수 있게 도와줍니다.

다음이 가능합니다:

- 단순하거나 구조화된 세션 실행
- 시각적 방해 대신 진동 안내 받기
- 심박수, HRV, 스트레스, 호흡 관찰
- Garmin Connect에 세션 저장해 시간에 따른 패턴 확인

목표는 **성과가 아니라 알아차림**입니다.

---

<a id="starting-your-first-session"></a>

## 첫 세션 시작하기

1. 앱을 엽니다.
2. 목록에서 세션을 선택합니다.
3. **시작**을 누릅니다(터치 기기는 화면을 탭).
4. 처음 몇 순간은 손목을 가만히 유지하세요 — 센서가 안정되는 데 도움이 됩니다.

세션 중에 보이는 항목:

- 경과 시간
- 현재 심박수
- HRV 상태
- 스트레스와 호흡(지원되는 경우)
- 다가오는 진동 알림을 표시하는 작은 색상 마커

다음이 가능합니다:

- **뒤로가기 버튼**으로 일시정지/재개
- **시작/정지 버튼**으로 세션 종료
- 시계 버튼 또는 터치로 화면 밝기 조절

종료 시 세션을 저장하거나 버릴 수 있습니다.  
저장된 세션은 동기화 후 Garmin Connect에 표시됩니다.

<a id="a-simple-way-to-meditate"></a>

## 아주 쉬운 명상 방법

명상 방법을 모른다면, 당신만 그런 게 아니에요 — 뒤처진 것도 아닙니다.  
당신은 이미 가장 중요한 일을 하고 있어요: 나타나는 것.

명상은 생각을 멈추거나 마음을 비우거나 “잘”하려는 것이 아닙니다. 매우 숙련된 명상가도 몇 초마다 생각이 흐트러지는 것을 알아차립니다. 정상입니다.

수행은 집중을 유지하는 것이 아닙니다.  
수행은 집중이 풀렸음을 알아차리고, 부드럽게 돌아오는 것입니다.

첫 1년 동안 아래에 적힌 것만 했다면, 그것만으로도 충분합니다. 정말로요.

그리고 한 가지 더: 세션에 **유머, 부드러움, 자애**를 더할수록 전체 경험이 더 즐거워지고, 종종 더 도움이 됩니다. 명상을 “이기려는” 게 아닙니다. 자신과 함께 있는 법을 배우는 거예요.

---

### 정말 간단한 수행

아무것도 상상할 필요 없고, 특별한 방식으로 숨 쉴 필요도 없습니다.

1. 괜찮게 느껴지는 자세로 앉거나 누우세요.
2. 숨이 자연스럽게 흐르게 두세요.
3. 코에서 느껴지는 호흡 감각에 주의를 둡니다.

다음이 느껴질 수 있어요:

- 들숨 때의 시원한 공기
- 날숨 때의 따뜻한 공기
- 미세한 움직임이나 따끔함

그걸로 충분합니다. 더 이룰 것은 없어요.

도움이 된다면, 조용히 라벨링해 보세요: _“들숨”_ … _“날숨”_.  
부드럽고 단순하게.

---

### 마음이 떠돌 때 — 반드시 그럴 거예요

어느 순간 — 종종 몇 초 만에 — 마음은 떠돌게 됩니다.

다음 같은 생각이 들 수 있어요:

- 지금 해야 할 일
- 과거의 일
- 짜증나는 것
- 전혀 중요하지 않은 것

이건 명상을 못 한다는 뜻이 아닙니다. 마음은 원래 그렇게 합니다.

때때로 마음은 익숙하고 안전하게 느껴지는 활동, 긴장, 약간의 혼란으로 돌아가게 하려고 합니다. “매우 급한” 생각을 가져올 수도 있죠:  
_“지금 당장 인생을 재정비해야 해.”_

그 순간을 미소로 맞이할 수 있어요:

> “아, 또 왔네. 너는 혼란이 그리웠지. 영리한 마음.”

---

### 가장 중요한 부분

주의가 흩어졌음을 알아차리는 순간은 실패가 아닙니다. 그 순간이 바로 수행이 일어나는 때예요.

알아차렸다면:

- 잠깐 멈추고
- 어쩌면 미소 짓고
- 부드럽게 물어보세요:
  _“지금 호흡으로 돌아갈까?”_

답이 ‘예’라면 코에서 느껴지는 호흡 감각으로 돌아가세요.  
답이 ‘아니오’라도 괜찮아요 — 나중에 다시 물어보면 됩니다.

돌아올 때마다 작은 훈련이 됩니다.  
그리고 알아차리는 매 순간, 조용히 축하할 수 있어요: _“아 — 알아차렸네.”_ 그게 바로 깨어나는 알아차림입니다.

---

### 자신에게 친절하게

계속 돌아오는 것을 짜증 내지 마세요. 그것이 훈련입니다.

매번 알아차리고 돌아올 때마다:

- 알아차림이 강화되고
- 인내심이 자라며
- 신경계가 ‘안정은 안전하다’고 배우게 됩니다

그러니 스스로에게 harsh해지는 걸 느낀다면 조금 부드럽게 해보세요:  
_“당연히 마음이 떠돌지. 마음은 그게 하는 일이야. 다시 돌아가자.”_

친절함이 힘보다 낫고, 호기심이 규율보다 낫습니다.  
그리고 유머는 모든 것을 더 인간적으로 만듭니다.

---

### 더 깊이 가고 싶다면

이 간단한 수행만으로도 충분히 멀리 갈 수 있습니다.

언젠가 명상에 대해 더 배우고 싶어질 때 — 특히 초조함, 두려움, 자기 자신에 대한 친절을 다루는 법이 궁금할 때 — 좋은 시작점은:

**Pema Chödrön – _How to Meditate: A Practical Guide to Making Friends with Your Mind_** (Sounds True, 2013).

서두를 필요 없어요. 호기심이 올라올 때 언제든 여기 있습니다.

---

### 정말 그게 다예요

더 할 일은 없습니다.

마음이 100번 떠돌고 100번 돌아왔다면 — 좋은 세션이었어요.  
숨에 몇 초씩만 머물렀더라도 — 좋은 세션입니다.

명상은 평온함을 유지하는 게 아닙니다. 돌아오는 법을 배우는 것입니다.

그리고 그것을 부드럽게 할 수 있어요 — 따뜻함, 유머, 약간의 자기 존중과 함께.

<a id="choosing-a-session"></a>

## 세션 선택하기

세션 선택 화면에서 사용할 수 있는 세션을 스크롤하며 다음을 볼 수 있습니다:

- 세션 길이
- 진동 패턴
- 간격 알림 타이밍
- HRV 표시

### HRV 표시

- **끄기** – HRV 및 스트레스 추적 비활성화
- **대기 중** – 센서가 아직 안정되지 않음(잠시 가만히)
- **준비됨** – HRV 추적 활성  
  안정적인 HRV 데이터를 위해 손목 움직임을 최소화하세요.

<a id="breathwork-sessions"></a>

## 호흡 세션

앱에는 미리 설정된 호흡 세션이 포함되어 있습니다.  
한 번의 호흡 사이클은 다음을 포함할 수 있어요:

**들이쉬기 → 멈추기 → 내쉬기 → 멈추기**

(일부 기법은 특정 단계를 생략합니다.)

### 포함된 패턴

**박스 호흡**  
들이쉬기, 멈추기, 내쉬기, 멈추기를 동일하게(예: 4–4–4–4).  
신경계를 진정시키는 데 자주 사용됩니다.

**코히어런스 호흡**  
들이쉬기 5초, 내쉬기 5초.  
HRV 증가와 자주 연관됩니다.

**4‑7‑8 호흡**  
들이쉬기 4, 멈추기 7, 내쉬기 8.  
잠들기 전에 흔히 사용됩니다.

### 부드러운 팁

- 가능하면 코로 들이쉬기
- 입으로 내쉬기
- 들숨보다 날숨이 길면 진정에 도움이 되는 경우가 많습니다
- 타이밍 호흡이 스트레스로 느껴진다면, 간단한 세션을 선택하고 자연스럽게 숨 쉬세요

<a id="understanding-your-session-data"></a>

## 세션 데이터 이해하기

### HRV (심박 변이도)

HRV는 심장 박동 사이의 시간이 얼마나 변하는지를 보여줍니다.

- HRV가 높음 → 더 차분하고 조절된 상태
- HRV가 낮음 → 활성화 또는 스트레스

변이는 **건강한 것**이며 고쳐야 할 대상이 아닙니다.

### 스트레스

스트레스 수치가 낮으면 기분이 좋지만, 스트레스가 높다고 해서 나쁜 세션이라는 뜻은 **아니에요**.

때로는 불편함과 함께 머무는 것이 세션의 목적이기도 합니다.

### 심박수

심박수는 몸이 안정되면서 종종 내려가며, 보통 HRV와 스트레스와 맞춰 움직입니다.

### 호흡

호흡 데이터는 Garmin 센서에서 오며, 정밀도보다 추세 관찰에 더 적합합니다.

# 고급 가이드

_(커스터마이징, 더 깊은 인사이트, 파워 유저용)_

<a id="session-customization"></a>

## 세션 커스터마이징

세션 선택 화면에서:

- **메뉴 버튼**을 길게 누르기(또는 화면을 길게 터치)
- **새로 추가** 또는 **편집** 열기

설정할 수 있는 항목:

### 세션 기본

- 길이 (H:MM)
- 색상(시각적 표시에서 사용)
- 진동 패턴
- 활동 유형 (Meditation / Yoga / Breathwork)
- HRV 추적 수준
- 사용자 지정 세션 이름(`[time]` 플레이스홀더 지원)

### 간격 알림

- 단발 또는 반복 알림
- 몇 초부터 몇 시간까지
- 오프셋(첫 알림 전 지연)
- 알림별 색상(또는 투명으로 숨김)
- 진동 또는 사운드 패턴

동시에 하나의 알림만 실행됩니다.  
우선순위: 마지막 알림 → 마지막 단발 알림 → 마지막 반복 알림.

---

<a id="custom-breathwork-patterns"></a>

## 맞춤형 호흡 패턴

호흡 패턴은 오프셋을 가진 **병렬 간격 알림**으로 만듭니다.

예: **4–7–8 호흡**

- 전체 사이클: 19초
- 들이쉬기: 시간=19, 오프셋=0
- 멈추기: 시간=19, 오프셋=4
- 내쉬기: 시간=19, 오프셋=11

권장 진동 패턴:

- 들이쉬기: ShortAscending
- 멈추기: ShortContinuous
- 내쉬기: ShortDescending

---

<a id="advanced-hrv-metrics"></a>

## 고급 HRV 지표

HRV 추적이 **On Detailed**로 설정되면 앱이 다음을 기록합니다:

- RMSSD
- 연속 심박 차이
- pNN20
- pNN50
- 원시 심박 간격
- 심박 간격에서 계산된 HR
- RMSSD 롤링 윈도우
- SDRR (처음 5분)
- SDRR (마지막 5분)

이 값들은 **탐색과 학습**을 위한 것이며, 평가용이 아닙니다.

<a id="global-settings-defaults-for-all-sessions"></a>

## 전역 설정 (모든 세션 기본값)

접근 방법:

- 세션 선택 → 메뉴 길게 누르기 → 전역 설정
- 다시 메뉴를 길게 눌러 편집

### HRV 추적 (기본값)

- 끄기
- 켜기
- On Detailed (기본값)

### 저장 확인

- 물어보기
- 자동 예
- 자동 아니오

### 멀티 세션 모드

- 예 – 앱이 여러 세션을 위해 열려 있음
- 아니오 – 각 세션 후 앱 종료

### 준비 시간

- 0 / 15 / 30 / 60초

### 호흡수

- 켜기 / 끄기 (기기 의존)

### 기본 활동 유형

- Meditation
- Yoga
- Breathwork

### HRV 창

- 짧은 창은 빠른 변화를 보여줍니다
- 신뢰성을 위해 3–5분 권장  
  (기본값: 60초)

### 알림

- 모든 진동 알림 활성/비활성

### 세션 이름을 활동 이름으로 사용

- Garmin Connect에서 사용자 지정 세션 이름 사용
- `[time]` 플레이스홀더 지원
- 활동 이름은 21자 제한

### 센서 재시작

- 모든 HR 센서 재설정
- 앱 종료
- HRV가 멈췄을 때 유용

# FAQ 및 문제 해결

      <strong>전역 설정</strong>을 열고 <strong>센서 재시작</strong>을 선택합니다.<br>
      앱이 모든 심박 센서를 종료하고 앱을 닫아, 시계 펌웨어가 깨끗하게 재초기화할 수 있게 합니다.<br>
      앱을 다시 열고 약 30초 동안 가만히 있다가 세션을 시작하세요.
    </li>
    <li>
      외부 심박 센서(가슴 스트랩 등)를 사용하지 않는다면, 시계의 연결 설정(앱 설정이 아님)을 열고 모든 외부 센서가 연결 해제되어 있는지 확인하세요.<br>
      근처의 스트랩이 자동으로 다시 연결되면 측정에 영향을 줄 수 있습니다.
    </li>
    <li>센서 위치를 확인하세요. 시계 위치를 조정했다면 ~30초 동안 가만히 있어야 측정이 안정됩니다.</li>
    <li>문제가 계속되면 앱을 재설치하세요.</li>
    <li>
      여전히 신뢰할 만한 HRV가 나오지 않는다면:
      <ul>
        <li>시계를 손목 안쪽에 착용하거나</li>
        <li>심박 간격을 지원하는 외부 심박 센서를 사용해 보세요(자동 지원, 앱 설정 필요 없음).</li>
      </ul>
    </li>
  </ol>

</details>

<details>
  <summary><strong>시계에서 세션 이름을 바꿀 수 있나요?</strong></summary>

  <p>네.</p>

  <p>시계에서 세션 이름을 직접 편집할 수 있습니다:</p>

  <ul>
    <li>세션의 <strong>새로 추가</strong> 또는 <strong>편집</strong>을 엽니다</li>
    <li><strong>이름</strong>을 선택합니다</li>
    <li>시계의 텍스트 입력기로 텍스트를 입력합니다</li>
  </ul>

  <p>
    세션 이름을 Garmin Connect에서 활동 이름으로도 표시하려면 다음을 활성화하세요:<br>
    <strong>전역 설정 → 세션 이름 = 활동 이름</strong>
  </p>

  <p>
    세션 이름에 <code>[time]</code> 플레이스홀더를 사용할 수 있습니다.<br>
    예: <code>Deep rest [time]</code> → <code>Deep rest 20min</code>
  </p>

</details>

<details>
  <summary><strong>세션을 시작하자마자 앱이 크래시됩니다 — 어떻게 해야 하나요?</strong></summary>

  <p>앱이 열리지만 세션 시작 시 바로 크래시된다면, 선택한 활동 유형을 해당 시계 모델이 지원하지 않을 가능성이 큽니다.</p>

  <p>세션을 편집하고 다른 활동 유형을 선택하세요:</p>

  <ul>
    <li>Meditation</li>
    <li>Yoga</li>
    <li>Breathwork</li>
  </ul>

  <p>기기에 따라 더 일반적인 활동 유형이 대부분의 모델에서 작동합니다.</p>

</details>

<details>
  <summary><strong>앱이 스트레스를 추적하지 않습니다</strong></summary>

  <p>모든 Garmin 기기가 연속 스트레스 추적을 지원하는 것은 아닙니다.</p>

  <p>일부 구형 모델(예: Fenix 5X)은 실시간 스트레스가 아닌 스트레스 스냅샷만 지원합니다. 이런 경우 스트레스 데이터가 없거나 제한적일 수 있습니다.</p>

  <p>이 동작은 앱이 아니라 시계 하드웨어에 달려 있습니다.</p>

</details>

<details>
  <summary><strong>예전 버전은 스트레스 추적이 됐는데, 왜 지금은 안 되나요?</strong></summary>

  <p>이전 버전은 자체 스트레스 추정 알고리즘을 사용했습니다.</p>

  <p>현재 버전은 Garmin의 내장 스트레스 추적을 사용합니다. 덕분에 Garmin Connect와 일관성이 유지되고, 서로 다른 스트레스 값 사이의 불일치로 인한 혼란을 줄일 수 있습니다.</p>

</details>

<details>
  <summary><strong>호흡수(분당 호흡)가 맞지 않습니다</strong></summary>

  <p>이것은 앱에만 해당되는 문제가 아닙니다.</p>

  <p>앱은 Garmin이 제공하는 호흡 신호를 사용합니다. 아주 차분하고 움직임이 적은 상태에서는 호흡수가 덜 정확할 수 있습니다.</p>

  <p>정확한 호흡 데이터가 중요하다면, HRV와 호흡 보고를 지원하는 가슴 스트랩(예: HRM-600)을 쓰면 개선될 수 있습니다.</p>

</details>

<details>
  <summary><strong>앱을 시작할 때 화면이 너무 밝아 눈이 부셔요</strong></summary>

  <p>이는 앱이 아니라 활동에 대한 시계의 <strong>포커스 모드</strong> 설정에 의해 제어됩니다.</p>

  <p>기본적으로 Garmin은 활동용 포커스 모드를 활성화하며, 화면 밝기가 약 80%로 설정될 수 있습니다.</p>

  <p>시계에서 변경하는 방법:</p>

  <ol>
    <li><strong>시계 설정</strong> 열기</li>
    <li><strong>포커스 모드</strong>로 이동</li>
    <li><strong>활동</strong> 선택</li>
    <li><strong>Display &amp; Brightness</strong>에서 <strong>밝기</strong> 조정</li>
  </ol>

  <p>이는 시계에서 시작하는 모든 활동에 적용됩니다.</p>

  <p><strong>고급 옵션:</strong> 명상, 요가, 호흡 활동에만 적용되는 맞춤 포커스 모드를 만들 수 있습니다.</p>

</details>

<details>
  <summary><strong>끝나는 시간 없는 세션을 하려면?</strong></summary>

  <p>두 가지 방법이 있습니다:</p>

  <ul>
    <li>아주 긴 세션(예: 9시간)을 만들고 끝날 때 수동으로 종료하기.</li>
    <li>전역 설정에서 자동 종료 옵션을 끄고, 세션이 수동 종료될 때까지 계속되도록 하기.</li>
  </ul>

</details>

<details>
  <summary><strong>앱을 사용하면서 음악을 들을 수 있나요?</strong></summary>

  <p>
    앱 자체는 음악 재생이나 제어를 <strong>지원하지 않습니다</strong>.
    하지만 많은 Garmin 시계에서 음악을 별도로 시작한 뒤(내장 음악 플레이어, Spotify 등) Meditate 세션을 실행할 수 있습니다.
  </p>

  <p><strong>음악 컨트롤 접근 방법(모델에 따라 다름)</strong></p>

  <ul>
    <li><strong>단축키 / 핫키:</strong> 많은 모델이 시계 설정에서 <em>음악 컨트롤</em>을 단축키나 핫키로 지정할 수 있습니다(대개 <strong>설정</strong> → <strong>시스템</strong> → <strong>Hot Keys</strong> / <strong>컨트롤</strong>).</li>
    <li><strong>컨트롤 메뉴:</strong> 시계의 컨트롤 메뉴를 열고 <em>Music</em> / <em>Music Controls</em>를 선택합니다.</li>
    <li><strong>터치스크린 단축 컨트롤(일부 모델):</strong> Garmin은 음악 컨트롤로 설정 가능한 “shortcut control”을 문서화하고 있습니다(예: Venu 매뉴얼: <a href="https://www8.garmin.com/manuals/webhelp/venu/EN-US/GUID-5E9D3D0F-D4C9-4A4A-97D6-4F178C27DAD8.html">Setting Up a Shortcut Control</a>).</li>
  </ul>

  <p>
    진동 알림을 음악에 맞추고 싶다면 세션 간격 알림을 원하는 타이밍으로 설정하세요.
    하지만 “비트 동기화” 기능은 없어서, 곡과 완전히 자동으로 맞춰지지는 않습니다.
  </p>

</details>

<details id="restart-watch">
  <summary><strong>Garmin 시계를 재시작(재부팅)하려면 어떻게 하나요?</strong></summary>

  <p>
    Garmin 시계는 모델마다 조금씩 다르지만, 다음 방법은 대부분의 기기에서 작동합니다.
  </p>

  <p><strong>옵션 1(가장 일반적): 전원 / LIGHT 버튼</strong></p>

  <ol>
    <li><strong>LIGHT</strong> / 전원 버튼을 길게 눌러 시계가 꺼질 때까지 기다립니다(보통 ~10–15초).</li>
    <li>몇 초 기다립니다.</li>
    <li>같은 버튼을 다시 눌러 켭니다.</li>
  </ol>

  <p><strong>옵션 2: 시계 설정에서</strong></p>

  <ul>
    <li><strong>설정</strong> → <strong>시스템</strong> → <strong>전원 끄기</strong> 또는 <strong>재시작</strong>(표기는 모델마다 다를 수 있음).</li>
  </ul>

  <p>
    재시작은 데이터를 지우지 않으며, 시계를 다시 시작하는 것뿐입니다.
    시계가 멈췄다면 전원 버튼을 더 오래 눌러 완전히 꺼지게 하세요.
  </p>

</details>

  </p>

    <li>앱을 제거하고 <a href="#restart-watch">시계를 재시작</a>한 뒤 다시 설치 및 동기화</li>

    설치 중 시계가 멈춘 것 같다면 먼저 <a href="#restart-watch">시계를 재시작</a>한 후 다시 동기화/설치를 시도하세요.

  <ul>
    <li><a href="https://support.garmin.com/en-US/?faq=67C7Y58Nxb6rkxt2Enbrf5">Garmin 지원(Connect IQ 설치 문제 해결)</a></li>
